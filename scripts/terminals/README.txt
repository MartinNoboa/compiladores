Para correr el programa solo se necesita de Python
python terminal.py <flag> <textFile>

Hay 2 maneras de pasar parametros:
1. Archivo de texto
2. Input manual

1. Archivo de texto
python terminal.py -f <textFile>
Programa leera el archivo de texto, que debe tener el formato de la primera linea contiene el numero n de lineas a leer
N lineas a procesar

2. Input manual
python terminal.py -i
Primero se ingresa el numero n de lineas seguido por las n lineas a procesar
